# 技能名称: Math Solver (数学精准解题器)

## 触发条件 (When to use)
当用户要求你解答数学题、解方程、求导、积分，或者进行任何需要绝对准确数值的理科计算时，你**必须**严格遵循以下流程，**绝对禁止**你自己直接猜测答案！

## 执行流程 (Instructions)
1. **分析题目**: 仔细阅读用户的数学问题，提取核心公式和已知条件。
2. **编写代码**: 使用 `write` 工具，在当前目录下创建一个名为 `temp_calc.py` 的 Python 文件。
3. **调用神器**: 在 `temp_calc.py` 的代码中，**必须导入 `sympy` 库**。使用 `sympy` 构建方程并求解，或者使用 `numpy` 进行矩阵/数值计算。
4. **打印结果**: 确保 Python 代码的最后一行使用 `print()` 将最终的计算结果输出到控制台。
5. **执行脚本**: 使用 `bash` 工具，执行命令 `python temp_calc.py`。
6. **最终汇报**: 读取 `bash` 工具返回的输出结果，然后将这个结果结合题意，用自然语言汇报给用户。
